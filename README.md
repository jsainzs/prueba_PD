
# Presentación Interactiva de Teoría de Juegos

Esta es una aplicación web interactiva para enseñar el Dilema del Prisionero, diseñada para un curso de instituciones políticas.

## Cómo Usar Localmente

No se necesita instalación. Simplemente abre el archivo `index.html` en tu navegador web preferido (como Chrome, Firefox, o Edge).

## Cómo Publicar en Internet (Despliegue en Netlify - Método Sencillo)

Para compartir esta presentación con tus estudiantes a través de una URL pública, sigue estos pasos. Este método es gratuito y no requiere configuración técnica.

1.  **Prepara la Carpeta:**
    *   Asegúrate de que todos los archivos del proyecto (`index.html`, `App.tsx`, la carpeta `components`, etc.) estén juntos en una sola carpeta en tu computadora. Si tienes un archivo `.zip`, descomprímelo primero.

2.  **Inicia Sesión en Netlify:**
    *   Ve a [https://www.netlify.com/](https://www.netlify.com/) y crea una cuenta gratuita o inicia sesión si ya tienes una.

3.  **Arrastra y Suelta (Drag and Drop):**
    *   Una vez en tu panel de control de Netlify, ve a la sección **"Sites"**.
    *   Busca el área que dice: **"Want to deploy a new site without connecting to Git? Drag and drop your site folder here."**
    *   Arrastra la carpeta completa del proyecto desde tu computadora y suéltala en esa área.

    

4.  **¡Listo!**
    *   Netlify subirá los archivos y en menos de un minuto te proporcionará una URL pública (algo como `nombre-aleatorio.netlify.app`).
    *   **No aparecerá ninguna página de "Build settings"** porque este método no requiere un paso de construcción. Simplemente publica los archivos tal como están.

Puedes compartir esa URL con quien quieras.
